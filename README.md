[Link on Figma](https://www.figma.com/file/r8BPYI4j22c0pTSvmDuWut/Messenger?node-id=1%3A616&mode=dev 'Link on Figma')

### Stack:

- Vite 4.4.9
- Node.js >14.0.0
- Handlebars
- Typescript 5.2.2
- SCSS

### Run the project

use `npm run dev` - for start development <br>
use `npm run start` - for build and start server
