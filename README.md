[Link on Figma](https://www.figma.com/file/r8BPYI4j22c0pTSvmDuWut/Messenger?node-id=1%3A616&mode=dev 'Link on Figma')

### Stack:

- Node.js >14.0.0
- Handlebars
- Typescript 5.2.2
- SCSS
- Vite 4.4.9

### Run the project

run `npm run dev` - developing mode <br>
run `npm run start` - server mode
