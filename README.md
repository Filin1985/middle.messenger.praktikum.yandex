### Stack:

- Node.js >14.0.0
- Handlebars
- Typescript 5.2.2
- WebSocket
- Mocha
- SCSS
- Vite 4.4.9

### Run the project

run `npm run dev` - developing mode <br>
run `npm run start` - server mode

### Test the project

run `npm run test` - server mode
